git .ignore使用摘要

git .ignore使用摘要

在專案目錄下建立.gitignore的檔案

內容為要排除的檔案

bin-debug/ (排除bin-debug這個資料夾) main.as (排除main.as這個檔案) *.log (排除副檔名為.log的檔案)

如果想要做ignore的檔案已經被commit進Repository

git rm -f 相關檔案

git commit

設定好.gitignore的檔案

git status看一下，確認沒有被列在Untracked files
